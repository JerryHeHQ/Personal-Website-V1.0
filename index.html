Recent Projects

Sertinary

Mobile Lifestyle App

Sertinary is a cross-platform Flutter & Firebase app designed to help users achieve their health and fitness goals. It features an enduring alarm that can only be turned off after the user completes a mini-game to prevent snoozing. To aid with diet awareness, it integrates a daily calorie and macro-nutrient tracker as well as meal creation and sharing.

Swap App

Android Trading App

Swap App is a native Android application designed for buying and selling goods. It features a smooth and intuitive user experience for post creation, browsing, trading, communication, and rating. For fast and efficient data synchronization, it implements realtime updates with Firebase Realtime Database.

GPU Deal Finder

Web Scraper For eBay

The GPU Deal Finder is a web scraper developed with JavaScript and NodeJS to acquire, process, sort, and display GPU listing data from eBay. It helped to purchase a 3080 for 18% less (~$115) when compared to average pricing at the time.
